FT Speech is a speech corpus created from the recorded and transcribed meetings of the Danish Parliament. To gain access to the corpus, please complete this Google form. A Kaldi recipe that trains a time-delay neural network on FT Speech can be found in this GitHub repo.

For more information on how FT Speech was created and evaluated, please refer to the paper FT Speech: Danish Parliament Speech Corpus, which has been accepted for presentation at INTERSPEECH 2020. Please cite it when using FT Speech material as:
